# ND+A Landing Page — Project Rules

## Tech Stack
- Next.js 15 (App Router), TypeScript, Tailwind CSS v4
- No database, no backend API
- Deploy target: Vercel

## Design Principles
- Exaggerated Minimalism: 과감한 여백, 초대형 타이포그래피
- Brand Colors: Red (#E5001E), Black (#0A0A0A), White (#FFFFFF)
- Font: Bebas Neue (display) + Source Sans 3 (body)
- Animations: CSS-only, 200-600ms, respect prefers-reduced-motion
- No emojis as icons — use Lucide React SVG icons

## Code Conventions
- Components colocated in app/_components/
- Tailwind utility-first, minimal custom CSS
- Korean comments allowed, English code
- All images must have alt text
- Semantic HTML (section, header, footer, nav, main)
- Mobile-first responsive design
